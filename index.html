Data structures:

Data structures in computers are storage units that are used for storing, organizing. processing and utilising data. There are different types of data types present, each of them having their own task. From maintaining the data related to function(call stack) to from handling website traffic(queue), data structures play a vital role.

Classification of Data structures

Data structures can widely be classified into Linear and Non-linear. These then can be further classified.
Linear Data structure: Data structure in which data elements are arranged linearly and each element is attached to the previous and next element.
Linear data structure can further be classified into Static and Dynamic.
     • Static data structure: Have fixed memory size. Ex: Array
      • Dynamic data structure: Size isnt fixed. Size can be updated during runtime. Ex. Stack, queue, linked list
Non-linear data structures: - Data structures in which elements aren't placed linearly or sequentially.
Ex:- tree, graph
